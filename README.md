# Simulación de Sistemas Interactivos — Bitácora

Bitácora de trabajo del curso **Simulación** (2026-20), donde documento mi proceso de aprendizaje: avances, experimentos, decisiones, dificultades, soluciones, evidencias y reflexiones a lo largo de las 8 unidades del curso.

📚 Curso guiado por el libro [*The Nature of Code*](https://natureofcode.com/)
🔗 Sitio del curso: [simulacion-2026-20](https://juanferfranco.github.io/simulacion-2026-20/)

---

## 📌 Sobre el curso

El curso busca desarrollar competencias para la **construcción de aplicaciones interactivas**, integrando conceptos de matemática y física para resolver problemas de entretenimiento digital, a través de la simulación computacional como forma de pensar, diseñar y construir sistemas generativos e interactivos.
